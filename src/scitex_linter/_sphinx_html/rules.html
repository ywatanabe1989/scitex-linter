Category FM: Figure/Millimeter (opt-in)

These rules detect inch-based matplotlib anti-patterns and suggest mm-based alternatives. FM rules require figrecipe or scitex[plt] to be installed, and must be explicitly enabled via enable = ["FM"] in config or SCITEX_LINTER_ENABLE=FM.

Rule

Severity

Description

STX-FM001

warning

figsize= detected — inch-based figure sizing is imprecise for publications

STX-FM002

warning

STX-FM003

warning

STX-FM004

STX-FM005

info

info
